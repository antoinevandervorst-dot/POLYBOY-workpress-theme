# POLYBOY - Wordpress Theme

Thème Wordpress personalisé réalisé dans le cadre d'un projet étudiant

## Description
Ce thème a été conçu pour le site de la console POLYBOY, dédiée au re-nouveau des jeux année 90

## Fonctionnalités
- Design responsive (Desktop)
- Personalisation CSS selon la charte graphique
- Structure basée sur l'arborescence du projet
- Compatibilité avec l'éditeur de bloc WordPress
